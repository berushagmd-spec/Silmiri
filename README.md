# Sil'mir Telegram Bot — простой вариант

Всё лежит в одном месте, без папок.

Файлы:

```text
bot.py
requirements.txt
.env.example
Сильмир.zip
```

## Запуск

1. Создай бота в Telegram через `@BotFather` и скопируй токен.
2. Переименуй `.env.example` в `.env`.
3. Вставь токен:

```env
BOT_TOKEN=твой_токен
```

4. Проверь, что `Сильмир.zip` лежит рядом с `bot.py`.
5. Установи зависимости:

```bash
pip install -r requirements.txt
```

6. Запусти:

```bash
python bot.py
```

Пока эта команда работает, бот онлайн.

## Проверка без Telegram

Можно проверить перевод прямо в терминале:

```bash
python bot.py --test
```

Можно заранее построить индекс словаря:

```bash
python bot.py --build-index
```

При первом запуске бот сам создаст файл `silmir.sqlite3`. Это не папка, а база-индекс для быстрого поиска по словарю.

## Как пользоваться в личке

Пользователь открывает твоего бота, нажимает `/start` и пишет русский текст.

Пример:

```text
я не вижу камень 😊
```

Ответ:

```text
yaś na velims kairən 😊
```

## Как пользоваться в группе

Добавь бота в группу. В группе бот отвечает только когда его упоминают или когда отвечают на его сообщение.

Пример:

```text
@username_бота я вижу камень
```

Ответ:

```text
Sil'mir:
yaś velims kairən
```

Можно писать и так:

```text
я вижу камень @username_бота
```

Обычные сообщения в группе бот игнорирует, чтобы не спамить.

Если бот в группе не видит сообщения с упоминанием, открой `@BotFather`, выбери бота через `/mybots`, зайди в `Bot Settings` → `Group Privacy` и выключи privacy mode. После этого лучше удалить бота из группы и добавить заново.

## Неизвестные слова

Смайлики и пунктуация остаются как есть. Если слова нет в словаре, бот покажет его в квадратных скобках, например `[телефон]`.


## Что значит "Читаю словарь" в логах

В этой версии бот запускается сразу и отвечает на `/start` и `/ping`, даже если индекс словаря ещё не готов. При первом переводе он один раз соберёт файл `silmir.sqlite3`. После этого следующие запуски будут быстрыми.

Если хочешь пересобрать словарь вручную, напиши боту `/reload`.

В группах бот отвечает только на сообщения с упоминанием, например:

```text
@username_бота я вижу камень
```

Если такие сообщения не доходят до бота, отключи privacy mode через BotFather: `/mybots` → бот → Bot Settings → Group Privacy → Turn off. Потом удали бота из группы и добавь заново.
