# Sil'mir Telegram Bot — простой вариант

Всё лежит в одном месте, без папок.

Файлы:

```text
bot.py
requirements.txt
.env.example
Сильмир.zip
```

## Запуск

1. Создай бота в Telegram через `@BotFather` и скопируй токен.
2. Переименуй `.env.example` в `.env`.
3. Вставь токен:

```env
BOT_TOKEN=твой_токен
```

4. Проверь, что `Сильмир.zip` лежит рядом с `bot.py`.
5. Установи зависимости:

```bash
pip install -r requirements.txt
```

6. Запусти:

```bash
python bot.py
```

Пока эта команда работает, бот онлайн.

## Проверка без Telegram

Можно проверить перевод прямо в терминале:

```bash
python bot.py --test
```

Можно заранее построить индекс словаря:

```bash
python bot.py --build-index
```

При первом запуске бот сам создаст файл `silmir.sqlite3`. Это не папка, а база-индекс для быстрого поиска по словарю.

## Как пользоваться в Telegram

Пользователь открывает твоего бота, нажимает `/start` и пишет русский текст.

Пример:

```text
я не вижу камень 😊
```

Ответ:

```text
yaś na velims kairən 😊
```

Смайлики и пунктуация остаются как есть. Если слова нет в словаре, бот покажет его в квадратных скобках, например `[телефон]`.
